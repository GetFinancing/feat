Things needed by Messy to run.

* Python 2.6
* Twisted >= 8.2.0
* CouchDB >= 0.10
* Paisley 0.3.1-feat.1 (There is a fork of paisley that can be found at:
                        https://github.com/f3at/paisley)
* simplejson
* Txamqp 0.3
* Amqp (required by txamqp)
* Rabbitmq-server >= 2.0


Simulator Gui dependences

* pydot
* pygtk
* python-configglue
